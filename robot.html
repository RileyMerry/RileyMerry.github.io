UGV Drone Project

Unmanned ground vehicle built around embedded systems and real-world integration.

This project combines Raspberry Pi hardware, C++ software, GPIO motor control, Linux services, browser-based operation, CAD design, and TCP/IP networking into a functional unmanned ground vehicle platform.

View System GitHub Repository

Target User & Mission Vision

The envisioned users include military, law enforcement, and emergency response teams requiring a compact ground vehicle for remote inspection and situational awareness.

The goal is not just movement, but demonstrating how embedded control, networking, and mechanical design integrate into a functional system.

CAD Model & Mechanical Design

The mechanical design was developed to support embedded hardware, motor drivers, and future sensor expansion. The focus was rapid prototyping and modularity.

Future improvements will focus on structural optimization, cable management, and sensor integration.

Networking Demonstration (TCP/IP)

The system operates using a client-server model where the Raspberry Pi acts as a server and the browser acts as the client.

The capture below shows a TCP SYN packet initiating a connection to the Raspberry Pi on port 8080.

This validates real transport-layer communication between the control interface and the embedded system.

Code Architecture

The project is structured using modular C++ components, Linux services, and a web interface that translates user input into motor control.
