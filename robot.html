UGV Drone Project

Unmanned Ground Vehicle built around embedded control and real-world system design.

This project explores how a Raspberry Pi, C++ software, GPIO motor control, Linux system services, browser-based control, CAD design, and networking concepts can come together into a working unmanned ground vehicle platform.

View Case Study GitHub Repository

Target User & Mission Vision

The envisioned target users for this platform are military, law enforcement, and emergency-response teams that may need a small ground vehicle for inspection, scouting, training, or remote observation in areas where sending a person first may be unnecessary or unsafe.

CAD Model & Physical Design

The first chassis design focuses on getting the system functional before optimizing for weight, material usage, sensor mounting, and user testing. The goal is to create a simple but reliable test platform that can evolve into a more refined final design.

Networking Demonstration

The robot uses a browser-based control interface where the Raspberry Pi acts as the server and the browser acts as the client. Wireshark captures can be used to show the TCP handshake and prove the client-server connection between the control page and the robot.

Code Architecture

The software is organized around C++ source files, CMake build structure, Linux scripts, GPIO control, front-end browser interaction, and backend logic that translates user input into motor control behavior.
